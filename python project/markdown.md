## 목록(list)

목록은 순서가 있는 목록과 순서가 없는 목록으로 구분된다.

1. 목록1
2. 목록2
   1. 목록2-1
   2. 목록2-2
3.  shift+tab

* 순서가 없는 목록
* 순서가 없음
  * tab하면 하위로
